1、使用 df 命令查看磁盘使用情况：
df -h
该命令会显示磁盘分区的使用情况，包括总空间、已用空间、可用空间以及挂载点等信息。选项 -h 可以以人类可读的方式显示磁盘大小，方便查看。

2、使用 lsblk 命令查看块设备信息：
lsblk
该命令会列出系统中的块设备信息，包括硬盘、分区、挂载点等。

3、使用 fdisk 或 parted 命令查看分区信息（需要管理员权限）：
sudo fdisk -l
或者
sudo parted -l
这些命令会显示磁盘的分区表信息，包括分区类型、起始扇区、大小等。

请注意，执行某些磁盘信息查询命令可能需要管理员（root）权限，所以在使用 sudo 时要小心谨慎。
