-- 查找关键字
vi sys-info.log


1、向前查找：

按下/键，然后输入要查找的文本，按下回车键。例如：/pattern。
Vi将定位到第一个匹配的文本行，并在光标下方高亮显示匹配的文本。按下n键可以继续向后查找下一个匹配。


2、向后查找：

按下?键，然后输入要查找的文本，按下回车键。例如：?pattern。
Vi将定位到最后一个匹配的文本行，并在光标下方高亮显示匹配的文本。按下n键可以继续向前查找前一个匹配。


3、忽略大小写
Vi编辑器的查找区分大小写。如果要进行大小写不敏感的查找，可以在查找命令前加上\c，表示忽略大小写；或者加上\C，表示区分大小写。

4、显示行号
临时显示行号：

按下冒号（:）键进入命令模式。
输入set number命令并按下回车键。例如：:set number。
Vi将在左侧显示每行的行号。
